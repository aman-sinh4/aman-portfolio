    Neumorphic and Glassmorphic Design

    Few days back I came across a twitter thread which helped me to design minimalistic forms with some cool UI styling. It is a trendy UI style nowadays. It is worth noting that neumorphism and glassmorphism is not a replacement for flat design style, but an addition to app use interface design.



    Neumorphism


    Steps to create a neumorphic form using few simple CSS steps :


    1. Create a container div and set background color darken than white . e.g #ecf0f3. Make sure that the color of body and container should be same.

    2. Also make sure left and top border of container should be brighter than right and bottom border. This will make the element looks like it has the light source illuminating form the top left corner of the screen.




    3. The input fields going to have a sunken effect. To achieve this we will use inset box-shadow. As light is coming from top-left corner, hence right and bottom border will be bright in this case because input fields have sunken effect.




    4. Create a button prefereably of same color as logo because this will give you a more fascinating look to your design. Addition to it we can add hover selector and apply box-shadow none when we mouse over it.

    Neumorphic design is all based on box-shadow property. we just nedd to remember two things down pointing backhand index-
    (a). Normal box shadow
    (b). Inset box shadow.

    You can also create neumorphic design using any color scheme you just need to remember that shades should not differ much.




    Glassmorphism


    Glassmorphism is a unified name for the popular frosted Glass design. It is trendy web design nowadays which looks super cool

    Let's create a glassmorphic form using few simple CSS steps :


    1. Starting of the basic structure, Just start with writing all necessary tags for form layout. You can any number of input field according to you.



    2. Create glass
    - Add thin white border
    - Low opacity white background color
    - Low opacity and large blur black shadow




    3. Style input field and button This is step same as step 2. We need to add glass look in the input field as well.




    4. Add some final detailing
    - One consistent text color
    - align "forgot password" and "signup"
